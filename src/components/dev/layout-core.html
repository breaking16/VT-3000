Layout Core

Базовий каркас сайту. Підключається автоматично.

1️⃣ Body states

body.lock    // блокує скрол (popup, menu)
body.loaded  // сайт завантажений

2️⃣ Wrapper

<body>
  <div class="wrapper">
    <header>...</header>
    <main>...</main>
    <footer>...</footer>
  </div>
</body>

✔ притискає footer
✔ фіксить sliders / overflow

3️⃣ gridContainer (рекомендовано)

.header__inner {
  @include gridContainer();
}
<div class="header__inner">
  ...
</div>

✔ явний
✔ не ламає grid
✔ працює з dynamic / fullpage
